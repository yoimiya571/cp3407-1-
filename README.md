Task One: Project Topic and Project Proposal

Our team has formed a project group with a maximum of 4 students. After discussion, we decided to work on the following project topic:

Project Title:
Read! - Reading Management System

Project Description:
Read! is a modern web-based reading management system developed using Django. The system helps users manage their reading activities, set annual reading goals, track reading progress, record completed books, manage personal profiles, upload avatars, and switch between different interface themes.

The project is suitable for students and book lovers who want to organize their reading records and monitor their reading habits more effectively.
